### 说明
#### Chrome(Chromium)浏览器(UC浏览器，360极速)
1. 菜单/更多工具/扩展程序(菜单/设置/管理扩展，菜单/工具/管理扩展)
2. 打开开发者模式
3. 加载已解压的扩展程序(选择文件夹)
#### FireFox浏览器
1. 菜单/附加组件/扩展
2. 设置图标/调试附加组件
3. 临时载入附加组件(选择manifest.json)
